# SafariXDebugToggle
Toggle XDebug in Safari 12 and beyond

# Motivation

Before macOS Mojave i used [XDebug Helper](https://github.com/mac-cain13/xdebug-helper-for-safari). But old extensions are disabled in Safari 12.
